博客搭建

发表于 2019-03-23
本文字数： 5.7k | 阅读时长 ≈ 5 分钟

本文采用github+hexo来搭建博客。

准备工作：

  1. 一个github账号
  2. 下载git
  3. 下载node.js

基础配置

本地：

本步骤所需时间较长，请耐心等待。

在本地创建一个新的文件夹（必须为新的/空的），名字自行定义。

打开文件夹后鼠标右键打开Git Bash。

输入npm install -g hexo下载Hexo

输入hexo init安装Hexo（此时文件夹下会陆续出现文件/文件夹）

输入npm install安装必要配置包

输入hexo s启动Hexo，打开浏览器输入地址localhost:4000查看

github：

在github上新建存储库你的github用户名.github.io

进入存储库后，打开这里：

找到这行并修改成这样：

然后在Git Bash中输入ssh-keygen生成SSH秘钥，并按3次回车，随后用记事本打开C:\Users\Administrator\.ssh\id_rsa.pub（本目录因电脑而异，大部分相同），把内容复制下来，打开https://github.com/settings/keys，新建一个SSH Key，粘贴key项目内（title随便填），最后按下add SSh key。

回到Git Bash，输入ssh -T git@github.com,若显示Hi!xxxx的字眼，则代表成功。

接下来输入npm install hexo-deployer-git --save安装上传工具deployer。

博客配置：

为区分根目录下的_config.yml和主题目录下的_config.yml，本文以“基础配置文件”代替根目录下的_config.yml，以“主题配置文件”代替主题目录下的_config.yml。

终于到最复杂配置博客的环节了，但其实认真读一遍说明你就会感到豁然开朗。

这里以NexT主题（V6.7.0）中的Gemini风格（就是这个博客的风格）的配置为例，同主题不同风格注意_config.yml中的设置限制（基本没有限制），其他主题参阅官网。

以下所有步骤均建议配置完成后输入hexo s启动本地浏览，进入localhost:4000验证。
